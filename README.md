# web-music-xmp

大四课程设计，基于SpringBoot+Vue开发的在线音乐网站。

## 在线体验
http://180.76.56.118:81/

## 技术栈
- 前端技术栈基于 Vue.js 开发前台后台
- 后端基于Spring  Boot + MyBatis 
- 数据库使用的是 MySQL。

## 系统功能
- 音乐播放
- 用户管理
- 歌曲管理
- 歌单管理
- 歌手信息
- 歌词显示
- 音乐播放
- 后台管理
